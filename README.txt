# Matija Fatić — GitHub Pages

## Kako da okačiš
1. Napravi novi GitHub repo, na primer: `matija-fatic-18`
2. Uploaduj sve fajlove iz ovog foldera u root repoa
3. Idi na **Settings > Pages**
4. Pod **Build and deployment** izaberi:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Sačuvaj i sačekaj par minuta

Posle toga ćeš dobiti link tipa:
`https://tvoj-username.github.io/matija-fatic-18/`

## Šta možeš lako da menjaš
- ime: u `index.html`
- datum i vreme: u `index.html`
- dugme za lokaciju: promeni `href`
- slika: zameni `assets/matija.jpg`
